# 🌻 My Emoji Playground

Welcome to **My Emoji Playground** — a fun and interactive React app that lets users create, customize, and play with emojis! 🎨😎

## 🚀 Features

- 🎭 Choose from a variety of emojis
- 🎨 Customize emoji colors
- 📏 Adjust emoji size with a slider
- 🔁 Reset to default settings
- 🖼️ Live preview of your emoji creation

## 🛠️ Tech Stack

- **React** (with Hooks)
- **CSS Modules** or **Styled Components** (depending on your styling choice)
- **FontAwesome** or Unicode for emoji rendering
